# Visual Studio Code Shortcuts para facilitar a vida

## Comandos Base

| Comando | Descrição |
|---|---|
| `Ctrl` + `Shift` + `P` | Abre todos os comandos disponíveis no VSCODE |
| `Ctrl` + `Q` | Abre as opções do Visual Studio Code (editor, extensões, git, etc...)
